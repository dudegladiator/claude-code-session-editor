cc-session

Interactive TUI + scriptable CLI for editing Claude Code session JSONL files.

Install

curl -fsSL https://get-claude-code-session-editor.harshiitkgp.in/install.sh | sh

macOS (arm64 / x86_64) and Linux (x86_64). Drops cc-session into /usr/local/bin or ~/.local/bin.

Other paths

cargo install cc-session
brew tap dudegladiator/tap && brew install cc-session
